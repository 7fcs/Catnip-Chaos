# Catnip-Chaos

Catnip Chaos is a Python arcade-library game inspired by Chrome's offline dinosaur runner, where you play as a cat dodging obstacles to rack up high scores. Collecting catnip too quickly triggers a chaotic "Catnip Craze" mode that scrambles the screen and makes the game harder, but rewards you with bonus points.

This is a group project for CS 122 (Advanced Python) at San Jose State University.


## Instructions

Install dependencies:
    `pip install -r requirements.txt`

Run:
    `python main.py`
